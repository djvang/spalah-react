# Spalah

## Setup

```
$ npm install
```

## Running in dev mode

```
$ npm start
```

## Running in production mode

```
$ npm run prod
```

## Build (production)

```
$ npm run build
```

## lint and stylelint
```
$ npm run lint
$ npm run stylelint
```

## test
```

$ npm run test
```
